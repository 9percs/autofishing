# AutoFisher for Hypixel

Simple Minecraft mod (.jar) for auto fishing on Hypixel.

Use smart or get banned lol.

## 🐟 How to use:

1. Put the `.jar` in your mods folder
2. Start Minecraft
3. Fish...
